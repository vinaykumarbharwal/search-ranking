# Contributing to RankSmart

We love your input! We want to make contributing to RankSmart as easy and transparent as possible.

## Development Process

1. Fork the repo and create your branch from `main`
2. Install development dependencies: `pip install -r requirements-dev.txt`
3. Make your changes
4. Run tests: `pytest tests/`
5. Ensure code formatting: `black src/ api/`
6. Submit a pull request

## Pull Request Guidelines

- Update the README.md with details of changes if needed
- Update the docs/ with any new functionality
- Add tests for new features
- Ensure test suite passes
- Use conventional commits format

## Report Bugs

Use GitHub issues to report bugs with:
- A quick summary
- Steps to reproduce
- Expected behavior
- Actual behavior
- Environment details
